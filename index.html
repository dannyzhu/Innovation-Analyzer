插件功能特点

使用 Kimi AI 工具分析、提取和记录知识信息，支持自定义多个Prompt，内置多个Prompt；也支持ChatGPT API。支持PDF、即时会话聊天、搜索。

  • 1. 免费： 基于Kimi AI平台，用户只需登录Kimi即可免费服务。
  • 2. 自定义Prompt：支持自定义Prompt，可定义多个Prompt。
  • 3. 支持 PDF：无需下载上传，浏览器中可直接访问 PDF。
  • 4. 跨浏览器兼容性：支持Chrome和Edge，同样也支持Arc浏览器。
  • 5. 对话和搜索：当前页即时对话聊天，支持搜索功能，支持PDF。
  • 6. Prompt库：提取数据、总结；提炼为PPT、微博、小红书风格、时间线。
  • 7. 快捷键操作：Alt+N 或 Shift+Alt+N ；支持右键菜单选择Prompt。
  • 8. 会话管理：自动保存至Kimi AI平台，可随时回顾和继续之前的对话。
  • 9. 视频网站：提取字幕内容，字幕全文翻译。
  • 10. ChatGPT API集成：支持ChatGPT API，以及兼容的其他大模型。

同时也支持 DeepSeek、智谱清言、零一万物、通义千问、Coze 机器人。效果对比Kimi最好，默认使用Kimi。

自定义 Prompt 功能，可定义多个 Prompt

可定义多个Prompt，每个Prompt可以配置System Prompt和User Prompt。

  • 单独配置，互不影响。
  • 随时修改，即刻生效。
  • 可选择缺省Prompt，打开即用。
  • KimiPlus 直接使用，方便快捷。
  • 安全：配置内容存储在您自己的电脑。
  • 支持 DeepSeek。
  • Kimi 和 DeepSeek 只需要你登录他们的网站即可使用，不需额外配置。

支持 Coze API 接入 Coze Agent，配置需要写 JSON 配置到 System Prompt 输入框即可（具体配置格式待补充）。

支持 PDF

支持 PDF 格式的文档，在浏览器中浏览PDF时可以直接在线提取内容并总结，无需下载。同时支持 KimiPlus。

  • 无需下载，直接在线浏览。
  • 支持多轮 Prompt。
  • 支持实时会话，多轮会话。
  • 支持搜索，相关内容及时搜索呈现。
  • 可跳转 Kimi 继续访问 PDF 文档。
  • 支持 Arc 浏览器。

无需下载和上传PDF，可在浏览器中直接使用，方便快捷。

会话聊天、即时搜索

可在当前浏览页面中通过会话和Kimi多次互动，即时搜索相关内容，提升效率。

  • 基于网页、PDF的内容进行会话交互。
  • 无需跳转，方便快捷。
  • 自动推荐5个可能的问题。
  • 根据输入的提示词自动匹配搜索。
  • 搜索的结果中包含来源链接。
  • 支持 Arc、Edge、Chrome 浏览器。

Kimi 会话交互功能，让用户在浏览网页、PDF时，通过与机器人进行会话交互，提升效率。

会话保留，KimiPlus 支持

使用 Kimi、智谱清言、零一万物、通义千问时，所有会话都将自动保存至AI大模型平台，用户可以随时回顾和继续之前的对话。

  • 自动保存会话内容。
  • 可直接使用 KimiPlus。
  • 直接跳转，方便快捷。
  • 单个网页在同一个会话中。
  • 支持 Arc 浏览器。
  • 所有功能无需特别配置，登录 Kimi、智谱清言、零一万物、通义千问即可。

使用DeepSeek、智谱清言、零一万物、通义千问等大模型平台也会保留会话。会话可长期保持，自动更新，不掉线。
