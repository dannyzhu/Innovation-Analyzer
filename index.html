插件功能特点

使用 Kimi AI 工具进行网页智能分析和输出，支持自定义Prompt，支持多Prompt，内置多个Prompt；也支持ChatGPT API。

  • 1. 免费： 基于Kimi AI平台，用户只需登录Kimi即可免费服务。
  • 2. 自定义Prompt：支持自定义Prompt，可定义多个Prompt。
  • 3. 支持 PDF：无需下载上传，浏览器中可直接访问 PDF。
  • 4. 跨浏览器兼容性：除了Chrome和Edge，同样支持Arc浏览器。
  • 5. KimiPlus 支持：可直接使用收藏的KimiPlus插件，且支持PDF。
  • 6. Prompt库：提取数据、总结；提炼为PPT、微博、小红书风格、时间线。
  • 7. 会话管理：自动保存至Kimi AI平台，可随时回顾和继续之前的对话。
  • 8. ChatGPT API集成：支持ChatGPT API，以及兼容的其他大型模型。

同时也支持 Coze 机器人，DeepSeek等。

自定义 Prompt 功能，可定义多个 Prompt

可定义多个Prompt，每个Prompt可以配置System Prompt和User Prompt。

  • 单独配置，互不影响。
  • 随时修改，即刻生效。
  • 可选择缺省Prompt，打开即用。
  • KimiPlus 直接使用，方便快捷。
  • 安全：配置内容存储在您自己的电脑。
  • 支持 DeepSeek。
  • Kimi 和 DeepSeek 只需要你登录他们的网站即可使用，不需额外配置。

支持 Coze API 接入 Coze Agent，配置需要写 JSON 配置到 System Prompt 输入框即可（具体配置格式待补充）。

支持 PDF

支持 PDF 格式的文档，在浏览器中浏览PDF时可以直接在线提取内容并总结，无需下载。同时支持 KimiPlus。

  • 无需下载，直接在线浏览。
  • 支持多轮 Prompt。
  • 支持 KimiPlus。
  • 可跳转 Kimi 继续访问 PDF 文档。
  • 支持 Arc 浏览器。

无需下载和上传PDF，可在浏览器中直接使用，方便快捷。

Kimi 会话保留，KimiPlus 支持

使用 Kimi 时，所有会话都将自动保存至Kimi AI平台，用户可以随时回顾和继续之前的对话。

  • 自动保存会话内容。
  • 可直接使用 KimiPlus。
  • 直接跳转，方便快捷；KimiPlus 也可跳转访问。
  • 单个网页在同一个会话中（KimiPlus 为单独会话）。
  • 支持 Arc 浏览器。
  • 所有功能无需特别配置，登录 Kimi 即可。

会话可长期保持，自动更新，不掉线。
